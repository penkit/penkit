# penkit tools

Library of lightweight pen testing tools.

## metasploit

Usage:

```bash
docker run --rm -it --net=host penkit/cli:metasploit
```


## sqlmap

Usage:

```bash
docker run --rm -it --net=host penkit/cli:sqlmap [options]
```

## wpscan

Usage:

```bash
docker run --rm -it --net=host penkit/cli:wpscan [options]
```
