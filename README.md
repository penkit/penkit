# penkit tools

Library of lightweight pen testing tools.

## metasploit

Usage:

```bash
docker run --rm -it --net=host penkit/tools:metasploit
```


## sqlmap

Usage:

```bash
docker run --rm -it --net=host penkit/tools:sqlmap [options]
```

## wpscan

Usage:

```bash
docker run --rm -it --net=host penkit/tools:wpscan [options]
```
